## 概要

このプライバシーポリシーは、koborin.ai（以下「当サイト」）がウェブサイト訪問時にどのような情報を収集し、利用し、保護するかについて説明します。

## 収集するデータ

### アナリティクス (Google Analytics 4)

当サイトでは、訪問者がどのようにウェブサイトを利用しているかを理解するために Google Analytics 4 を使用しています。このサービスは以下の情報を収集します：

- 閲覧したページと各ページでの滞在時間
- おおよその地理的位置（国・都市レベル）
- デバイスとブラウザの情報
- 参照元（当サイトへのアクセス経路）

これらのデータは匿名化され、コンテンツとユーザー体験の改善のみに使用されます。Google Analytics はこの情報を収集するために Cookie を使用します。Cookie の使用に同意しない場合、アナリティクスのトラッキングをオプトアウトできます。

詳細については [Google のプライバシーポリシー](https://policies.google.com/privacy?hl=ja)をご覧ください。

### コメント (Giscus)

当サイトのコメントシステムは [Giscus](https://giscus.app/) を使用しており、GitHub Discussions を利用しています。コメントを投稿する際：

- GitHub アカウントで認証します
- GitHub のユーザー名とアバターがコメントと共に表示されます
- コメントは当サイトの GitHub リポジトリの Discussions に保存されます

当サイトが GitHub のパスワードにアクセスすることはありません。詳細については [GitHub のプライバシーステートメント](https://docs.github.com/ja/site-policy/privacy-policies/github-privacy-statement)をご覧ください。

## Cookie

以下の目的で Cookie を使用しています：

| Cookie | 目的 | 有効期間 |
|--------|------|----------|
| `cookie_consent` | Cookie の設定を記憶 | 1年 |
| `_ga`, `_ga_*` | Google Analytics トラッキング（同意時のみ） | 2年 |

Cookie の設定は、ブラウザの Cookie をクリアするか、ブラウザのプライバシー設定を使用することでいつでも管理できます。

## あなたの権利 (GDPR)

欧州連合にお住まいの場合、以下の権利があります：

- 当サイトが保持する個人データへのアクセス
- 不正確なデータの修正を要求
- データの削除を要求
- データ処理に異議を唱える
- いつでも同意を撤回

これらの権利を行使するには、以下の連絡先までご連絡ください。

## サードパーティサービス

当サイトでは以下のサードパーティサービスを使用しています：

- **Google Analytics** - ウェブアナリティクス
- **Giscus** - コメントシステム（GitHub 経由）

各サービスには、それぞれのデータ取り扱いを規定する独自のプライバシーポリシーがあります。

## データ保持

- アナリティクスデータは Google Analytics のデフォルト設定に従って保持されます
- コメントは著者またはリポジトリオーナーが削除するまで残ります
- Cookie 同意設定はブラウザにローカルに保存されます

## ポリシーの変更

このプライバシーポリシーは随時更新される場合があります。重要な変更がある場合は、このページの上部にある「最終更新日」を更新してお知らせします。
